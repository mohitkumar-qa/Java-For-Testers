# Java For Testers

This repository contains Core Java programs written for software testing 
and automation preparation.

## Topics Covered
-  Basic Concepts of Java Programming
-  Strings & String methods
-  Objects-Oriented Programming
-  Special Class in Java    
-  Regex for Testers
-  Arrays
-  Collections : List, Maps, Sets
-  Exception Handling

## Tools Used
- Java 17
- Eclipse IDE
- Git & GitHub
